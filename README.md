# Online Advertising Revenue Prediction

##  Project Overview
This project predicts online advertising revenue using Machine Learning.

It includes data preprocessing, model training, evaluation, and deployment using Streamlit.

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

---

##  Model Used
- Decision Tree Regressor
- Scikit-learn Pipeline for preprocessing
- Model saved using pickle

---

##  Model Performance
- Train R2 Score: 0.98
- Test R2 Score: 0.93

---

##  Web Application
The Streamlit app allows users to input campaign details and predicts the expected advertising revenue instantly.

---

##  Project Files
- `app.py` – Streamlit application
- `model.pkl` – Trained machine learning model
- `requirements.txt` – Required libraries
- `README.md` – Project documentation
